# 拾词 · 本地英语单词学习工具

这是一个面向初中生的本地英语背单词工具。教材照片、词库和学习记录都只保存在电脑本机。

## 在 Mac 上使用

1. 安装 [Node.js](https://nodejs.org/) 22 或更高版本。
2. 将整个文件夹复制到 Mac 的可写目录，例如“文稿”。不要只复制 `index.html`。
3. 第一次使用时，在终端执行一次：`chmod +x start.command`。
4. 以后双击 `start.command`，浏览器会自动打开学习工具。
5. 使用结束后，在网页“数据与设置”中点击“安全退出”。

如果 macOS 阻止打开启动文件，可右键点击 `start.command`，选择“打开”。

## 使用顺序

1. 在“添加单词”中上传清晰的 JPG、JPEG 或 PNG 教材词表照片，也可以手工录入。
2. 校对识别出的英文和中文释义，确认后存入词库。
3. 首页会显示词库中全部未学习新词。每天第一次打开时，自己选择今天学习几个，也可以选择 0；当天关闭再打开仍会保留这批计划新词。
4. 从首页开始学习：新词先认读，再根据中文和发音拼写。
5. 答错的词需要随后连续答对两次，并在 1、3、7、14、30 天后复习。
6. 定期在“数据与设置”中导出备份。

## 自主复习

“自主复习”只使用已经学习过的单词，不会提前消耗未学习新词。

- 可以手动勾选单词，也可以从易错词、已掌握或全部已学词中自动抽取。
- 自由练习支持认读和拼写；答错会计入易错统计，但不改变原来的复习日期。
- 正式复习使用拼写检测，答题结果会推进或重置正式复习阶段。
- 手动拼写答错后仍会稍后回来，需要连续答对两次。

照片识别使用已打包在工具内的 PaddleOCR PP-OCRv5 模型，不会把照片上传到网络。系统会自动判断表格方向；横置照片会自动转正并复核一次，多列表格会自动排除音标、词性、表头和页脚，再配对英文与中文释义。只有最佳识别结果仍然过少或配对率偏低时，才会显示旋转后重试。

教材中标在英文词首的 `*` 星号属于课外延伸词标记，识别时会自动去除，不会作为单词的一部分存入词库。

## 支持范围

- 照片格式：JPG、JPEG、PNG；不支持 HEIC。
- 发音：使用 macOS 和浏览器安装的英文语音，不联网下载音频。
- 数据：保存在本文件夹的 `data` 目录。移动工具时应复制整个文件夹。
- 词库：单一总词库，允许重复英文，每条记录独立学习。

## 开发检查

Windows 可运行 `start-windows.bat`。开发环境安装依赖后，可执行 `npm run build` 重新生成 OCR 浏览器运行包，执行 `npm test` 运行自动检查；普通使用者不需要安装这些开发依赖。
