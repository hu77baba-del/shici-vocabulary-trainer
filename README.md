# 拾词 · 本地英语单词学习工具

> 每天一点，记得更久。

![拾词英语学习插画](public/assets/home-english-study-banner.png)

拾词是一款面向初中生和家庭使用场景的本地英语单词学习工具。它把教材词表识别、每日新词计划、拼写练习、间隔复习和自主复习放在一个简洁的网页界面里运行。

这个程序是我专门用来给正在上海读初一的小朋友学习英语的。

照片、词库、学习记录和备份都保存在用户自己的电脑中。程序不依赖在线账号，也不会把教材照片上传到网络。

## 功能亮点

- **每日新词自己决定**：首页显示词库中全部未学习新词，每天可以选择学习 `0` 到全部数量，不再固定每天 10 个。
- **当天计划不会乱变**：确认后固定当天词条，中途退出或刷新仍会继续同一批；开始学习后可以增加，但不能减少。
- **记忆节奏复习**：单词会在 1、3、7、14、30 天后再次出现，完成全部周期后进入“已掌握”。
- **错词强化循环**：拼写错误的单词会稍后重新出现，并需要连续答对两次才能完成本轮。
- **自主复习**：支持手动勾选，以及从易错词、已掌握或全部已学词中自动抽取。
- **自由与正式两种模式**：自由练习支持认读和拼写，不改变正式复习进度；正式复习通过拼写结果推进或重置复习阶段。
- **词库批量管理**：在“我的词库”中勾选多个词条，或全选当前搜索、筛选结果后一次删除；删除时会同步清理相关学习记录和当天计划引用。
- **教材词性贯穿学习**：单词可保存 `n.`、`v.`、`adj.` 等教材词性和多词性；认读、每日拼写、到期复习和自主复习都会显示，词组不显示空占位。
- **本地教材 OCR**：内置 PaddleOCR PP-OCRv5，可识别 JPG、JPEG、PNG 教材词表，自动处理横置照片、透视倾斜、多列表格、长短释义、斜杠同义短语、音标、词性、表头和页脚，并把词性附加到已有中英文配对。
- **连续学习反馈**：星光勋章会记录连续学习天数，在数值增长时播放一次轻量庆祝动画。
- **备份与恢复**：可导出完整备份，也能在恢复前自动保留当前数据。

## 快速开始

### Windows

1. 安装 [Node.js](https://nodejs.org/) 22 或更高版本。
2. 下载或克隆整个仓库，不要只保存 `index.html`。
3. 双击 `start-windows.bat`。
4. 浏览器会自动打开学习工具。
5. 使用结束后，在“数据与设置”中点击“安全退出”。

### macOS

1. 安装 [Node.js](https://nodejs.org/) 22 或更高版本。
2. 将整个项目放到可写目录，例如“文稿”。
3. 第一次使用时，在终端执行：

   ```bash
   chmod +x start.command
   ```

4. 以后双击 `start.command` 启动。
5. 如果 macOS 阻止打开，可右键点击 `start.command` 并选择“打开”。

`v1.1.1` 已在真实 Mac 设备上确认能够正常启动和运行；`v1.3.1` 尚未在 Mac 上验证。

## 推荐使用流程

1. 在“添加单词”中上传清晰的教材词表照片，或者手工录入英文和中文释义。手工录入支持原来的 `英文<TAB>中文释义`，也支持 `英文<TAB>词性<TAB>中文释义`。
2. 校对英文、词性和中文释义；单词缺少词性时会提示但允许保存，词组无需填写词性。
3. 回到首页，选择今天准备学习的新词数量，也可以选择 `0`。
4. 新词先完成认读，再根据中文释义和词性进行拼写；拼写测验不会朗读英文，也不提供发音按钮。
5. 按首页提示完成到期复习，或者进入“自主复习”自由安排练习。
6. 定期在“数据与设置”中导出备份。

## 自主复习说明

“自主复习”只会使用已经学习过的单词，不会提前消耗未学习新词。

- **手动选择**：搜索、筛选并勾选需要复习的词。
- **自动抽取**：从易错词、已掌握或全部已学词中填写数量后随机抽取；易错词优先选择错误次数较多的内容。
- **自由认读**：通过“认识 / 不认识”进行自评。
- **自由拼写**：自动判断拼写是否正确；错误计入易错统计，但不改变复习日期和阶段。
- **正式拼写**：正确会推进正式复习阶段，错误会重置为学习中，并进入连续答对两次的强化循环。

完成一组自主复习后，会计入当天连续学习记录。

## 本地 OCR 与隐私

照片识别使用随项目打包的 PaddleOCR.js 和 PP-OCRv5 mobile 模型，运行时不需要 CDN，也不会上传照片。

系统会自动判断词表方向。横置照片会先物理转正再复识别，多列表格会排除音标、表头和页脚，先配对英文与中文释义，再沿每条配对行附加位于两列之间的词性。只有 `0` 组可靠中英文配对时，页面才会建议旋转后重试。

教材中标在英文词首的 `*` 星号通常表示课外延伸词，识别时会自动去除，不会作为单词的一部分保存。

## 数据与支持范围

- 数据默认保存在项目文件夹的 `data` 目录中。
- `data/app-data.json`、自动备份和教材样例已被 `.gitignore` 排除，不会进入 Git。
- 移动工具时应复制整个文件夹，并建议先导出一份备份。
- 支持 JPG、JPEG、PNG；目前不支持 HEIC。
- 认读卡片和词库手动发音使用操作系统及浏览器已有的英文语音，不联网下载音频；拼写测验不提供发音提示。
- 当前使用单一总词库，允许保存重复英文，每条记录独立维护学习状态。
- 旧备份没有词性字段时仍可恢复；新备份会完整保留 `partOfSpeech` 和 `partOfSpeechNeedsReview`。
